# tdd-chess
Practise TDD, DDD; learn how to use Immutable objects; by implementing a chess game

Run the nl.roka.app.Main class.
Valid input:
  q - quit
  r - reset game
  "xx xx" - make a move 


The console board sucks because it doens't align properly.
